# RethinkDB Store for SenecaJS

This is the driver for RethinkDB storage in the Seneca framework.

## Tests

1. Start RethinkDB on `localhost`
2. Open RethinkDB console at http://localhost:8080/
3. Create a database named `test`, if it does not exist
4. Create a table named `foo` in database `test`
5. Create another table named `moon_bar` in database `test`
6. Run `npm test`
